# 硬件连接说明

本项目用于 WT9932P4-TINY / ESP32-P4 开发板 + 7 英寸 MIPI-DSI 触摸屏，实现 LVGL 触摸屏虚拟手柄界面。

## 硬件清单

- WT9932P4-TINY / ESP32-P4 开发板
- 7 英寸 1024 x 600 MIPI-DSI LCD 屏幕
- EK79007 LCD 驱动芯片
- GT911 触摸芯片
- 屏幕转接板
- USB-C 数据线

## 屏幕转接板连接

| 屏幕适配板 | WT9932P4-TINY |
| --- | --- |
| 5V | 5V |
| GND | GND |
| PWM | GPIO26 |
| LCD_RST | GPIO27 |
| INIT_TP | GND |

## MIPI-DSI 排线

把 LCD 的 FPC 排线连接到开发板的 `MIPI_DSI` 接口。

注意：

- 排线方向必须正确
- 使用两面同面的 FPC 连接线
- 插入后锁紧排线座
- 如果屏幕只亮背光但没有图像，优先检查排线方向

## USB 连接

使用 USB-C 数据线连接开发板到电脑。

要求：

- 数据线必须支持数据传输
- 电脑设备管理器里应出现 COM 串口
- 如果只供电、不出现 COM 口，请更换数据线或安装串口驱动

## 上电顺序

1. 接好屏幕转接板供电线
2. 接好 MIPI-DSI 排线
3. 用 USB-C 连接电脑
4. 打开开发板电源开关
5. 在设备管理器中确认 COM 串口
6. 烧录固件

## 功能验收

硬件连接成功、固件烧录成功后，应满足：

- 屏幕显示虚拟手柄 UI
- 触摸屏可以点按虚拟按键
- 多指触摸时，多个虚拟控件可以同时响应
- 串口日志能看到 `GamePad UI created`
- 串口日志能看到 `HID transport boundary started`

## 排查顺序

### 屏幕完全不亮

1. 检查 USB-C 供电
2. 检查开发板电源开关
3. 检查屏幕转接板 5V/GND
4. 检查背光 PWM 接线

### 屏幕亮但没有 UI

1. 检查 MIPI-DSI 排线方向
2. 检查 LCD_RST 是否接到 GPIO27
3. 确认屏幕为 EK79007 1024 x 600
4. 查看串口日志是否有 `Display started`
5. 查看串口日志是否有 `GamePad UI created`

### UI 显示正常但触摸无反应

1. 检查触摸排线方向
2. 确认触摸芯片为 GT911
3. 查看串口日志是否有 GT911 初始化错误
4. 检查触摸连接线是否松动

### 电脑不识别串口

1. 更换 USB-C 数据线
2. 更换 USB 口
3. 安装 CH343/CH34x 串口驱动
4. 重启电脑或重新插拔开发板
